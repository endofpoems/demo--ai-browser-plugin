核心功能

四大核心能力

从总结到可视化，覆盖网页学习的完整链路。

📄

页面智能总结

一键调用 AI 对当前网页内容进行系统性归纳，围绕"是什么、为什么、使用场景、原理机制"四个维度展开，生成精炼的结构化总结。

🗺️

树状思维导图

基于页面总结自动生成树状思维导图，支持缩放、拖拽平移和节点折叠展开。技术类内容按"是什么/为什么/场景/优缺点"五维度分类着色。

💬

划词链式解释

选中任意文本即可获得 AI 一句话解释。支持链式追问——对解释内容再次划词，弹出新消息框深入解析，实现层层递进的知识探索。

📥

Markdown 一键导出

将页面总结和思维导图一键导出为 Markdown 文件，方便保存到本地笔记工具或分享给他人，构建个人知识库。

使用流程

三步开始学习

无需复杂配置，安装即用。

1

配置 API Key

在插件设置中填入兼容 OpenAI 协议的 API 地址和密钥，支持任意兼容服务商。

2

一键总结

打开任意网页，点击插件图标或按 Alt+S，AI 将自动分析页面内容并生成结构化总结。

3

深度探索

展开思维导图可视化知识结构，使用划词功能（Alt+W 开关）深入理解任意概念，导出 Markdown 存档。

快捷键

高效操作

键盘快捷键让你无需离开页面即可完成核心操作。

Alt + S 触发页面总结
Alt + W 开启 / 关闭划词总结
安装指南

获取插件

Chrome 开发者模式加载，一分钟完成。

📦 手动安装步骤

  1. 从 GitHub 仓库 克隆或下载项目源码，进入项目根目录执行 npm install && npm run build
  2. 打开 Chrome 浏览器，地址栏输入 chrome://extensions/ 并回车
  3. 打开右上角 开发者模式 开关
  4. 点击 加载已解压的扩展程序，选择项目中的 dist 文件夹
  5. 点击浏览器工具栏中的插件图标，进入设置配置 API Key 即可使用

支持所有兼容 OpenAI Chat Completions 协议的 API 服务
